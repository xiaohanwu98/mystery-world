Group Hello Final Project README


1. Open the file "group_Hello_finalproject.pde".


2. Please download “controlP5” and “minim” from the library;


3. Click "run" on the top of the screen. The welcome page of the game will pop out.


4. Please read the game rules before start the game. Game rules are written in the Instruction part of the GUI system. 


5. Use the mouse to click on buttons; use the keyboard to control the character.


6. The project description is written in the finalproject_description.txt.
